# Installing the launcher

The mod is distributed exclusively through [mod launcher / updater](https://github.com/descatal/ModdedBoost).

Please download the latest launcher [here](https://github.com/descatal/ModdedBoost/releases/latest):

Click on the `.msi` file link under the "Assets" section of the latest release to start the download.

Once the download is complete, open the file and install the application.
